## Tech Stack
- **Frontend**: React, JavaScript (ES6+)
- **Styling**: Vanilla CSS3 (Variables, Flexbox/Grid), Glassmorphism, Animations
- **Motion**: Framer Motion
- **Build Tool**: Vite
- **Routing**: React Router v6

## Running Locally

1. **Clone the repository**
   ```bash
   git clone https://github.com/YusufTheDev/YusufTheDev.github.io.git
   cd YusufTheDev.github.io
   ```

2. **Install Dependencies**
   ```bash
   npm install
   ```

3. **Start Development Server**
   ```bash
   npm run dev
   ```

## Deployment

This project uses `gh-pages` for deployment.
- **Source Code**: `main` branch
- **Live Site**: `gh-pages` branch

To deploy changes:
```bash
npm run deploy
```
