Focus Forge

FocusForge is a productivity tool created during a build-a-thon with the goal of supporting individuals with ADHD. Designed with an intuitive interface and customizable features, FocusForge helps users stay organized, manage tasks, and maintain focus. The challenge of the build-a-thon was to create a product in under 24 hours, and due to this, a static game was made using html, css and js.

Meteorite Defender

"Meteor Defender" is an engaging game developed using Python and Pygame. Players control a spaceship and must defend it from meteorites by using missiles. The game features multiple levels and an endless mode, offering both structured challenges and limitless gameplay. With its intuitive controls and dynamic gameplay, "Meteor Defender" provides an immersive and exciting experience for players of all ages.

Web Pong

WebPong is a dynamic multiplayer Pong game developed with JavaScript, HTML, CSS, and Socket.io. This project features robust server architecture, enabling numerous players to compete against each other simultaneously in real-time. Experience the thrill of this retro game directly in your browser with smooth, interactive gameplay.
