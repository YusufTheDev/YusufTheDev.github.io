Automated Crypto Trading Bot

This project is a 24/7 automated cryptocurrency trading bot built with Python, AWS, CCXT, and the Coingecko and Kraken APIs. It executes trades based on Bollinger Bands (BB) and Relative Strength Index (RSI) indicators, using dynamic thresholds for optimal performance. The bot integrates position sizing, stop-loss, and take-profit mechanisms for risk control. It was deployed on AWS EC2 and backtested across multiple markets to achieve a balanced strategy between profitability, drawdown, and risk management.

Ascend Limits Music Bot

Ascend Limits is a next-generation Discord music bot built with Python and discord.py, delivering seamless and high-fidelity audio playback through FFmpeg and Lavalink integration. It features real-time filters, loop modes, dynamic queue control, and an interactive slash-command system for an intuitive user experience. Designed with scalability and reliability in mind, Ascend Limits provides smooth performance and modular architecture for maintainability and future expansion.
